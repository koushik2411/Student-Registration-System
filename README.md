Project name: 'Student Registration System'

Tech Stack & Tools Used:
  1. HTML: Designed the basic structure of the document which includes header, form, and table.
  2. TailwindCSS: Done the modification and styling of elements such as text colors, font size, elements sizes, responsiveness, etc.
  3. CSS: Added some styles such as backgrounds and responsiveness using Media Queries.
  4. JavaScript: Added all the logics and functions on the project.
  5. LocalStorage: Used to save files so that they remain in the system even after refreshing the page.
  6. Font Awesome Icons: Used icons from Font Awesome to add some aesthetic appearance in the respective labels of the form.
  7. Git & Github: Version control and storing project online.
  8. VS Code: For writing the entire codes.

Short Description: This is a simple Student Registration System where one can register a student and store the data in localStorage so the data can't be lost after refreshing the page. Here one can edit or delete a student's data after adding them. It also shows the list of the registered students.
